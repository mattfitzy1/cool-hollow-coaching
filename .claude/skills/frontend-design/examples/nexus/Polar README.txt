## HOW TO USE

You have two ways to use this template:

Option 1 – Remix in Aura (Recommended)

* Remix directly in Aura.
* Edit layouts, content, and styles visually.
* No coding required.
* Ideal for fast customization.

Option 2 – HTML Editing

1. Open the HTML files in your browser.
2. Edit using a code editor (VS Code recommended).
3. Replace content, images, and branding.
4. Deploy to your hosting provider.
