# Copy

Copy the most recently drafted text (email, message, bio, or other output) to the clipboard using pbcopy.

If the user specifies what to copy, copy that specific content. If no argument is given, copy the last piece of drafted content from the conversation.

## Rules
- Use pbcopy to copy to clipboard
- Do not add any formatting or markdown — plain text only
- Confirm with a short message when done (e.g. "Copied.")
- If it's unclear what should be copied, ask
