# System: 12-Month Impact Map

> Takes an uploaded initiative list (rated against the Strategy Razor: core customer fit, unfair advantage fit, impact, effort) and outputs the filtered 3-5 priorities plus a focus score. In-program tool for Milestone 2 of Business Without You.

## Architecture

```
[CSV/Excel initiative list upload] --> [analysis.py: load_initiatives + build_impact_map] --> [Streamlit report in browser]
```

## Key Files

| File | Purpose |
|------|---------|
| `apps/impact-map/app.py` | Streamlit web app: file upload, renders the kept/cut lists, serves the branded template via a download button |
| `apps/impact-map/analysis.py` | Core logic: applies the razor, ranks by impact-for-effort |
| `apps/impact-map/Cool_Hollow_Coaching_Milestone_2_Impact_Map_Template.xlsx` | Branded client-facing input template, generated by `apps/_shared/generate_templates.py` |
| `apps/impact-map/sample_initiatives.csv` | Internal dev-testing fixture, not client-facing |
| `apps/impact-map/requirements.txt` | Pinned dependency versions |

## How It Works

1. User uploads a CSV or Excel sheet: one row per initiative, with `initiative`, `core_customer_fit` (1-5), `unfair_advantage_fit` (1-5), `impact` (1-5), `effort` (1-5), rated from the owner's Strategy Razor draft, Core Customer Decree, and Unfair Advantage list.
2. `load_initiatives()` normalizes the sheet and clamps every rating to the 0-5 range.
3. `build_impact_map()` applies the razor: an initiative passes only if core_customer_fit, unfair_advantage_fit, and impact are all 3 or higher. Anything that fails is cut regardless of how appealing it looks.
4. Initiatives that pass are ranked by an impact-for-effort score (`impact - effort/5`) and the top 5 are kept. If fewer than 3 pass the razor, the model fills up to 3 from the best of what failed, so the map is never empty.
5. The app shows the focus score (percentage of the original list cut), the kept list with reasoning, and an expandable list of everything cut with the specific razor gap that sank it.

## Configuration

No API keys needed. Runs entirely locally on Python + Streamlit + pandas + openpyxl.

## Common Operations

**Start the app:**
```bash
cd apps/impact-map
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

**Test the scoring logic directly:**
```bash
source .venv/bin/activate
python3 -c "
import pandas as pd
from analysis import load_initiatives, build_impact_map
initiatives = load_initiatives(pd.read_csv('sample_initiatives.csv'))
print(build_impact_map(initiatives))
"
```
