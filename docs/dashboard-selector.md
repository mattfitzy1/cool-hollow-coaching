# System: Monday Morning Dashboard

> Takes an uploaded candidate metric list across five categories (cash, sales, delivery, customer, team) and selects one metric per category with a red/yellow/green threshold, flagging up to two leading indicators. In-program tool for Milestone 3 of Business Without You.

## Architecture

```
[CSV/Excel candidate metrics upload] --> [analysis.py: load_metrics + build_dashboard] --> [Streamlit report in browser]
```

## Key Files

| File | Purpose |
|------|---------|
| `apps/dashboard-selector/app.py` | Streamlit web app: file upload, renders the five-metric dashboard, serves the branded template via a download button |
| `apps/dashboard-selector/analysis.py` | Core logic: selects one metric per category, sets thresholds |
| `apps/dashboard-selector/Cool_Hollow_Coaching_Milestone_3_Dashboard_Template.xlsx` | Branded client-facing input template, generated by `apps/_shared/generate_templates.py` |
| `apps/dashboard-selector/sample_metrics.csv` | Internal dev-testing fixture, not client-facing |
| `apps/dashboard-selector/requirements.txt` | Pinned dependency versions |

## How It Works

1. User uploads a CSV or Excel sheet of candidate metrics: one row per metric, with `category` (cash, sales, delivery, customer, or team), `metric_name`, `current_value`, `target_value`, `direction` (higher_better or lower_better), `leading` (yes/no), and `impact` (1-5, how directly the metric reflects whether the business is healthy).
2. `load_metrics()` normalizes and validates the sheet, rejecting unrecognized category or direction values.
3. `build_dashboard()` picks the single highest-impact candidate per category (ties broken toward leading indicators), giving exactly five metrics.
4. Each selected metric gets a status: green if current value is at or past target, yellow if within a 10% band of target, red beyond that, direction-aware so lower_better metrics aren't penalized for being low.
5. Among the five selected, up to two marked `leading` are flagged as the ones to check first, ranked by impact.
6. The app reports the five-metric dashboard with color status and leading flags, and errors clearly if a category has no candidates at all.

## Configuration

No API keys needed. Runs entirely locally on Python + Streamlit + pandas + openpyxl.

## Common Operations

**Start the app:**
```bash
cd apps/dashboard-selector
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

**Test the selection logic directly:**
```bash
source .venv/bin/activate
python3 -c "
import pandas as pd
from analysis import load_metrics, build_dashboard
metrics = load_metrics(pd.read_csv('sample_metrics.csv'))
print(build_dashboard(metrics))
"
```
