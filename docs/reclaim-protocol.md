# System: 15-Hour Reclaim Protocol

> Takes an uploaded Liberation Audit (one-week time log sorted into owner_only / delegate / automate / eliminate) and outputs a named, deadlined reclaim list. In-program tool for Milestone 1 of Business Without You.

## Architecture

```
[CSV/Excel time log upload] --> [analysis.py: load_time_log + build_protocol] --> [Streamlit report in browser]
```

## Key Files

| File | Purpose |
|------|---------|
| `apps/reclaim-protocol/app.py` | Streamlit web app: file upload, renders the reclaim list, serves the branded template via a download button |
| `apps/reclaim-protocol/analysis.py` | Core logic: normalizes the time log, scores and ranks candidates |
| `apps/reclaim-protocol/Cool_Hollow_Coaching_Milestone_1_Reclaim_Protocol_Template.xlsx` | Branded client-facing input template, generated by `apps/_shared/generate_templates.py` |
| `apps/reclaim-protocol/sample_time_log.csv` | Internal dev-testing fixture, not client-facing |
| `apps/reclaim-protocol/requirements.txt` | Pinned dependency versions |

## How It Works

1. User uploads a CSV or Excel time log: one row per task, with `task`, `hours_per_week`, `category` (owner_only, delegate, automate, eliminate), and optionally `documented` (yes/no), `trained_replacement` (yes/no), `risk` (low/medium/high). Missing optional columns default to no / medium.
2. `load_time_log()` normalizes and validates the sheet, rejecting unrecognized category values.
3. `build_protocol()` splits owner_only tasks (left alone) from delegate/automate/eliminate candidates. Each candidate gets an urgency score: `hours_per_week + readiness (documented/trained bonus) - risk penalty`.
4. Candidates are ranked by urgency and split into three deadline tiers (this week / within 2 weeks / within 30 days) by rank position, with plain-English reasoning per item (what's blocking it, what the risk means for how it should be handed off).
5. The app shows the total reclaimable hours and the remaining owner-only load, then the deadlined list grouped by tier.

## Configuration

No API keys needed. Runs entirely locally on Python + Streamlit + pandas + openpyxl.

## Common Operations

**Start the app:**
```bash
cd apps/reclaim-protocol
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

**Test the scoring logic directly:**
```bash
source .venv/bin/activate
python3 -c "
import pandas as pd
from analysis import load_time_log, build_protocol
log = load_time_log(pd.read_csv('sample_time_log.csv'))
print(build_protocol(log))
"
```
