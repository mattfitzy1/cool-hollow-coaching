# System: Bottleneck Breakthrough Plan

> Takes an uploaded Constraint Identification worksheet and ranks every candidate constraint by combined impact, names the single binding constraint, and builds an effort-adjusted automation hit list. In-program tool for Milestone 6 of Business Without You.

## Architecture

```
[CSV/Excel constraint worksheet upload] --> [analysis.py: load_constraints + build_breakthrough_plan] --> [Streamlit report in browser]
```

## Key Files

| File | Purpose |
|------|---------|
| `apps/bottleneck-breakthrough/app.py` | Streamlit web app: file upload, renders the ranked plan and hit list, serves the branded template via a download button |
| `apps/bottleneck-breakthrough/analysis.py` | Core logic: scores and ranks constraints, builds the automation hit list |
| `apps/bottleneck-breakthrough/Cool_Hollow_Coaching_Milestone_6_Bottleneck_Breakthrough_Template.xlsx` | Branded client-facing input template, generated by `apps/_shared/generate_templates.py` |
| `apps/bottleneck-breakthrough/sample_constraints.csv` | Internal dev-testing fixture, not client-facing |
| `apps/bottleneck-breakthrough/requirements.txt` | Pinned dependency versions |

## How It Works

1. User uploads a CSV or Excel sheet: one row per candidate constraint, with `constraint_name`, `process`, `frequency_per_week`, `hours_lost_per_occurrence`, `downstream_impact` (1-5), `automatable` (yes/no), `automation_effort` (1-5, 1 is trivial, 5 is hard).
2. `load_constraints()` normalizes and validates the sheet, rejecting unrecognized `automatable` values.
3. `build_breakthrough_plan()` computes `weekly_hours_lost = frequency_per_week x hours_lost_per_occurrence`, then a `constraint_score = weekly_hours_lost x downstream_impact`.
4. Every constraint is ranked by `constraint_score`. The top-ranked one is named the binding constraint, the one to break first.
5. Constraints marked `automatable` get a separate `automation_roi = weekly_hours_lost / automation_effort` score, ranked independently so the easiest, highest-payoff automations surface first regardless of where they rank on raw impact.
6. The app shows the binding constraint headline, the full ranked list, and the automation hit list.

## Configuration

No API keys needed. Runs entirely locally on Python + Streamlit + pandas + openpyxl.

## Common Operations

**Start the app:**
```bash
cd apps/bottleneck-breakthrough
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

**Test the ranking logic directly:**
```bash
source .venv/bin/activate
python3 -c "
import pandas as pd
from analysis import load_constraints, build_breakthrough_plan
constraints = load_constraints(pd.read_csv('sample_constraints.csv'))
print(build_breakthrough_plan(constraints))
"
```
