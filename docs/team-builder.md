# System: Build the Team

> Takes an uploaded role inventory and delegation map, ranks roles by delegation gap, builds an outcome-based hiring template per role, and (with an optional candidate list) scores candidates on a five-trait leadership scorecard. In-program tool for Milestone 7 of Business Without You, the last of the seven milestone models.

## Architecture

```
[CSV/Excel role inventory upload] --> [analysis.py: delegation scores + hiring templates]
[CSV/Excel candidate list upload, optional] --> [analysis.py: leadership scorecard]
                                              --> [Streamlit report in browser]
```

## Key Files

| File | Purpose |
|------|---------|
| `apps/team-builder/app.py` | Streamlit web app: two uploads (one optional), renders all three outputs, serves the branded template via a download button |
| `apps/team-builder/analysis.py` | Core logic: delegation scoring, template building, candidate scoring |
| `apps/team-builder/Cool_Hollow_Coaching_Milestone_7_Team_Builder_Template.xlsx` | Branded client-facing input template (Roles + Candidates tabs), generated by `apps/_shared/generate_templates.py` |
| `apps/team-builder/sample_roles.csv` | Internal dev-testing fixture, not client-facing |
| `apps/team-builder/sample_candidates.csv` | Internal dev-testing fixture, not client-facing |
| `apps/team-builder/requirements.txt` | Pinned dependency versions |

## How It Works

1. User uploads a role inventory: one row per role, with `role_name`, `current_delegation_level` (1-5), `target_delegation_level` (1-5), `key_outcome_1/2/3`, `decision_rights`, `success_metric`.
2. `build_delegation_scores()` ranks every role by `target_delegation_level - current_delegation_level`, biggest gap first, so the owner knows which role to fix before the next.
3. `build_hiring_templates()` turns each role's own key outcomes, decision rights, and success metric into an outcome-based hiring template, replacing a generic job description with what the role actually needs to own.
4. If the user also uploads a candidate list (`candidate_name`, `role_name`, and five 1-5 ratings: `ownership`, `communication`, `judgment`, `coachability`, `results_track_record`), `build_leadership_scorecard()` averages the five traits per candidate, ranks candidates within their role, and flags the top candidate per role.
5. The app shows the delegation-gap ranking, an expandable hiring template per role, and the leadership scorecard if candidates were provided.

## Configuration

No API keys needed. Runs entirely locally on Python + Streamlit + pandas + openpyxl.

## Common Operations

**Start the app:**
```bash
cd apps/team-builder
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

**Test the scoring and templating logic directly:**
```bash
source .venv/bin/activate
python3 -c "
import pandas as pd
from analysis import run_team_builder
roles = pd.read_csv('sample_roles.csv')
candidates = pd.read_csv('sample_candidates.csv')
print(run_team_builder(roles, candidates))
"
```
